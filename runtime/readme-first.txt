0. I AM NOT RESPONSIBLE FOR ANY DAMAGES, LOSS OF TIME, OR BAD JUDGMENT

1. backup your phone!  If this does not work, you can then recover using your backup

2. java should be in your path so if you type java from the command line it runs

3. from the cmd prompt change to the folder your reading this readme file from

4. Run ResignUpdate.bat or ResignUpdate.sh
4a. Click the ... button and pick your ROM zip file.
4b. Click the Resign button and wait for it to finish working.

5. Once complete you will have a new file with a name simular to the one precess, but will have "-toggle2g.resigned.zip" at the end of the filename.

6. Do a FULL WIPE and flash resigned ROM in the recovery.
